ACCESS THE DEMO

The Broadleaf Evaluation tool provides a comprehensive platform for exploring an eCommerce solution. It operates independently, requiring no reliance on third-party services. This self-contained environment facilitates easy customization, allowing users to tailor their unique eCommerce setup with varying levels of personalization.

 

Whether implementing minor modifications or extensive customizations, this tool enables a flexible and straightforward evaluation experience without external dependencies. The framework includes 30+ microservices containing extensible commerce service components built on Java and Spring as well as a centralized metadata-driven back office admin framework built in React.
